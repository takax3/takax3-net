DojiOnee は、 Discord 上で利用できる AI Bot 。

コンセプトは「ドジっ娘お姉ちゃん」。  
現代の LLM はどうしても会話のコンテキストを完全に汲むことができない、最新の知識やコアな知識に弱い、等の課題がある。  
そこで、あえてキャラクター性として含めることにより、ポンコツであることをユーザに不満に思わせないことを目論んだ。

ユーザの使い方によって、サーバ内に不和を引き起こしたことを除いてはおおむね好調。  
みんなに楽しんでもらうための Bot として作ったのにね……。悲しいね。

## 実装のポイント
- **コスト重視の設計** - 外部 API を用いて推論するため、コストが雪だるま式に膨れ上がる恐れがあった。回数制限・トークン削減などによりコストを削減。
- **知識の補強** - Function Callingにより、 LLM 自身が検索が必要かを判定。検索により得られたコンテキストを含め、回答を補強。
- **親しみやすさ** - 名前の記憶や、ギルド内での頻出単語等を分析。また、明示的に呼ばなくてもランダムに反応することで、ユーザへの親しみやすさを向上。

ポンコツですけれど、可愛がってあげてください。
