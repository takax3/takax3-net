# About

ふんわりしたアイデアや課題を、要件整理・実装・運用まで落とし込むエンジニアです。

業務ではIT領域で、要件整理、設計、実装、インフラ、ネットワークまわりに関わっています。  
個人では、AI Bot、取引戦略の検証ツール、Webサイト、業務自動化のような小さな仕組みを作っています。

## 得意なこと

- ふんわりした要望を整理し、実装可能な形に落とすこと
- Pythonによる小さな自動化・データ処理
- API連携や通知Botの作成
- Dockerを使った小規模なアプリ運用
- Cloudflare / AWS を使ったWeb公開
- Linux / ネットワークまわりの調査・構成検討
- 作ったものを継続運用できる形に整えること
