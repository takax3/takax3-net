# SPECIFICATION — takax3-net

サイトの方向性とスコープの正本。変更は人間ゲート G2（agents.md）。

## 目的

恋ヶ窪タカノ（taka / たかにゃん）のポートフォリオ兼技術ブログ。「何を作っている人か」が一目で伝わり、制作物と記事に自然に誘導すること。

- サイト名: **恋ヶ窪タカノの実装室**
- タグライン: **「あいまいなアイデアを、動く仕組みに。」**
- 公開先: https://takax3.net/ （Cloudflare Pages、main への push で自動デプロイ）

## デザインの方向性

「キャラクター＆クラフト系」— 丸み・暖かみ・遊び心。テック系ダーク調・ミニマル系は不採用（決定済み: docs/decisions.md）。

- 配色: ライト = クリーム `#FBF9F4` 背景 + ブルー `#3B6FD4`（キャラの髪色由来）/ ダーク = ネイビー `#10182A` 背景 + `#8FB0F5`
- フォント: 見出し M PLUS Rounded 1c / 本文 Zen Kaku Gothic New
- 角丸 14-16px のカード、柔らかい影、ピル型タグ
- キャラクター（透過立ち絵）をヒーローとフッターに配置

## スコープ

- 自己紹介（トップ + About）
- 実績・プロジェクト一覧と詳細（`src/content/projects/`）
- 技術ブログ（`src/content/blog/`、draft 除外）
- OGP・RSS・sitemap などの公開品質向上（Phase 4）

## 非スコープ

- コメント機能・お問い合わせフォームなどの動的機能
- CMS・管理画面（コンテンツは Markdown を直接編集）
- 多言語対応（日本語のみ)
- アクセス解析以上のトラッキング

## 成功条件

- スマホ・デスクトップの両方で崩れなく表示される
- `npm run build` が警告なく通り、Cloudflare Pages のデプロイが成功する
- トップページを見て「キャラクター＆クラフト系」の雰囲気とタグラインが伝わる（人間が目視判断）
